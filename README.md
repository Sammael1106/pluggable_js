# PluggableJs

Convention of how to load javascript only in certain pages based on controller and action parameters.

## Installation

Add this line to your application's Gemfile:

    gem 'pluggable_js'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install pluggable_js
